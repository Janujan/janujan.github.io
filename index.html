Timeline

Peraso Technologies

At Peraso Technologies, I joined the Analog Evaluation Team as an RF Engineer. I had the oppurtunity to characterize and fully test their next generation WiGig chipset. I was able to interface with RFIC directly and investigate any problems that arise. I developed an in-house ATE program to characterize thousands of chips and I designed a GUI that shift workers can use to properly sort chips.

Peraso Technologies

As I enjoyed my previous term at Peraso Technologies, I decided to go back once again! On my second term, I got to delve into the baseband side of the Peraso WiGig solution. I developed code to interface with new test equipment and improve the inhouse ATE experience. I also analyzed extensive data collected on chip performance and presented it to the design team for review.

Cognitive Systems Corp

As a DSP Developer at Cognitive Systems, I developed the LTE stack and implemented a frequency offset algorithm to improve over-the-air(OTA) performance. I really enjoyed developing driver level code to a Cognitive System's own powerful R10 chip. I also improved the custom Python testing framework for the R10 solution.

BlackBerry Limited

As a sensor design engineer at BlackBerry, I had the oppurtunity to both characterize and design the main TOF sensor of the BlackBerry Radar project. I developed C# code to interface with a sensor qualification board and read out the results from testing. After the initial tests, I analyzed the data captured to provide insights on the best way to use the sensor to pull informaiton regarding it's environment.
